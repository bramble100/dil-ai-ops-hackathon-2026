# Annual Report 2023 (Geschaeftsbericht 2023)

**File:** `raw/RheinmetallAGAnnualReport2023.pdf` (283 pages, German)

## Key Facts

- **Reporting period:** FY2023
- Revenue: 7,176 MioEUR (+12.0% YoY)
- Operating Result: 918 MioEUR (margin 12.8%)
- EBIT before PPA: 968 MioEUR (new metric)
- Order Intake: 11,843 MioEUR (more than doubled from 5,629m)
- Rheinmetall Backlog: 38,290 MioEUR
- DAX index entry

## Notable Data

This report introduced the expanded order metrics framework (Booked Business, Frame Nomination, Frame Utilization, Rheinmetall Nomination, Rheinmetall Backlog) that better captures the multi-layered defense procurement pipeline.

Balance sheet total jumped from 8.1bn to 11.7bn -- reflecting M&A activity and defense capacity investment. Net financial debt moved from -426m to -1,063m.

Historical comparison tables in this report provide the complete 2017-2023 financial time series used across the wiki.
